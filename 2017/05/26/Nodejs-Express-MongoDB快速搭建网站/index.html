  • Node.js
  • Express框架(通过npm安装)
  • Moment.js——处理时间格式(通过npm安装)
  • mongoose——mongoDB数据库插件(通过npm安装)
  • underscore——mongoDB相关插件，可实现数据更新(通过npm安装)
  • cookie-parser——解析cookie(通过npm安装)
  • express-session——处理session(通过npm安装)
  • connect-mongo——用mongodb做会话的持久化(通过npm安装)
