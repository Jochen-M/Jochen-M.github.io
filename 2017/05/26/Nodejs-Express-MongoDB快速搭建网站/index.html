Nodejs + Express + MongoDB快速搭建网站

基本架构

  • 后台：
    • Node.js
    • Express框架(通过npm安装)
    • Moment.js——处理时间格式(通过npm安装)
    • mongoose——mongoDB数据库插件(通过npm安装)
  • 数据库：
    • mongoDB
  • 前端：
    • jade模板引擎(通过npm安装)
    • Bower(通过npm安装)
    • jQuery(通过Bower安装)
    • Bootstrap(通过Bower安装)

注意事项及技巧

  • 使用bower安装依赖的库和文件时，应事先在项目根目录建立.bowerrc文件，并在文件中添加配置信息，表示库和文件的安装目录：

  • 使用bower init和cnpm init生成配置文件。
