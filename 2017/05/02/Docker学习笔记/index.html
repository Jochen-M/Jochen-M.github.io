Ubuntu + Docker 学习笔记

什么是Docker

  • Docker is the world’s leading software container platform.
  • Docker公司开发，开源，托管在github
  • 跨平台，支持Windows、Macos、Linux

Docker思想

  • 集装箱
  • 标准化，包括运输方式、存储方式、API接口标准化
  • 隔离（类似于虚拟机，不过更加轻量）
